## Featured Project: MoodLens

**MoodLens** is an **AI-powered mental wellness tracker** helping users reflect on mood, stress, and well-being.  

- Tracks daily mood & stress with an accessible interface  
- Provides **AI-driven insights** and personalized coping exercises  
- Built with **React, Tailwind CSS, Python, FastAPI**, integrates **Google Gemini API**  
- Uses **AWS Polly** for text-to-speech & Web Speech API for voice input  
- Data stored locally to ensure **privacy & security**
